In the homepage there is a menu that has four choices:

  • Single Player
  • Tow Players
  • Options
  • Statistics

The Single Player redirects the user to a page that allow the user to play verses the computer. Two Players let two player play in the same computer. The options let you choose the board theme, turn the sound on/off and choose the difficulty of the game. The Statistics shows how many games played, wins, tie, lost and the percent of winning. Also it provide a bar chart shows these information. In addition, there is a reset buttton to delete the statistics.

When you click either single palyer or two player the page shows an empty board. the user need to click at the top of the column to insert a disc. The first player to match four in a row wins!

There is Main Menu button to redirect the to game menu page which is the website homepage. New game button reset the board, redraw the board with empty cells. There is two text field to show how many the user win(You) and the computer or other player (Visitor).
